# 📌 سامانه NTSW (National Trade Single Window)

## معرفی سامانه
سامانه جامع تجارت (NTSW) یا «پنجره واحد تجارت»، بستر یکپارچه‌ای برای مدیریت فرآیندهای تجارت داخلی و خارجی ایران است.  
بخش‌های اصلی سامانه:
- تجارت داخلی
- تجارت فرامرزی
- سامانه شناسه کالا
- سامانه انبارها
- سامانه ارزی (نیما)
- سامانه مجوزها
